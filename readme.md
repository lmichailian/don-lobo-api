Instalar proyecto
```
$ npm install
```


#### <i class="icon-file"></i> Correr las migraciones

```
$ ./ace migration:run
```

#### <i class="icon-folder-open"></i> Generar los seeds

```
$ ./ace db:seed
```

#### <i class="icon-pencil"></i> Iniciar el servidor

```
$ npm run serve:dev
```

#### <i class="icon-trash"></i> Iniciar el cron Job

```
$ ./ace scheduler:run
```
